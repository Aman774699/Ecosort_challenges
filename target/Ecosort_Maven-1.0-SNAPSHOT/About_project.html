About Our Project

The purpose of the Waste Sorting and Recycling Gamification App is to transform the often mundane task of waste sorting and recycling into an enjoyable and rewarding experience. By gamifying the process and providing users with educational resources and incentives, the app aims to increase recycling participation rates, reduce contamination in recycling streams, and ultimately contribute to a cleaner and more sustainable environment.



The app is developed using Java for core logic implementation, JDBC for backend development,DataScience For model Predicting and simulation of API requests. Additionally, image recognition technology may be integrated using libraries such as OpenCV to enable real-time identification of recyclable items from images captured by users.
